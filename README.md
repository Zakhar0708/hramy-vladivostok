# Православные храмы Владивостока

Информационный веб-сайт для дипломной работы по специальности "Информационные системы и программирование". Содержит каталог ключевых храмов Владивостокской епархии с поиском, фильтрами и интерактивной картой.

![Скриншот сайта](screenshot.png) <!-- Добавь скриншот сайта -->

## Возможности
- Поиск храмов по названию и адресу
- Фильтры: соборы / храмы / все
- Интерактивная карта Yandex Maps с метками
- Адаптивный дизайн для мобильных

## Установка и запуск
1. Скачай все файлы (`index.html`)
2. Открой `index.html` в браузере
3. Замени `ТВОЙ_API_КЛЮЧ` на ключ Yandex Maps (бесплатно на api.yandex.ru)
4. Сайт готов!

**Демо:** [твоя-ссылка.github.io/hramy-vladivostok](https://твоя-ссылка.github.io)

## Технический стек
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Карта:** Yandex Maps API
- **Данные:** JSON-объекты (10 храмов Владивостока)

## Источники данных
- [Официальный сайт Владивостокской епархии](https://www.vladivostok-eparhia.ru)
- [2GIS Владивосток](https://2gis.ru/vladivostok)
